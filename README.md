概要
================================

MUI v7 のサンプルです。

以下の要素を含みます:

- コンポーネント分割:
  - Atomic Design, Colocation
- スタイリング:
  - MUI v7
- ルーティング:
  - React Router v7
- 状態管理:
  - Context
- バックエンド（未実装）:
  - JSONPlaceholder
- その他:
  - レスポンシブ, ダークモード
